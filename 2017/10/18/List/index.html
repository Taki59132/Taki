CH2, p22 Phone Directory Application 完整程式碼

  1. obj == this 此行成立於當一個物件是reference from另一個物件時，因此，他們的object ID將會一樣。
  2. getClass() 將會回傳當前class名稱，此例對this.getClass()->DirectoryEntry
  3. Problem : No enclosing instance of type E is accessible. Must qualify the allocation with an enclosing instance of type E(egxnew A() where x is an instance of E).
    Solution : 將public class改為public static class.
    原因: 在Java中，類別中的靜態方法不能直接調用動態方法。只有將某個內部類修飾為靜態類，然後才能夠在靜態類中調用該類的成員變量與成員方法。
